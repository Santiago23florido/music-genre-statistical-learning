\section{Introduction}
This project studies automatic music genre recognition from acoustic descriptors stored in \texttt{data/Music\_2026.txt} and \texttt{data/Music\_test\_2026.txt}. The labeled file contains 7,773 observations, 191 numerical predictors, and one target variable named \texttt{GENRE}, while the holdout file contains 3,798 observations described by the same 191 predictors.

The five classes present in the labeled dataset are \textit{Blues}, \textit{Classical}, \textit{Jazz}, \textit{Pop}, and \textit{Rock}. The report is organized as follows. Section II presents the data description and preprocessing. Section III covers the unsupervised analysis. Section IV focuses on binary classification for \textit{Classical} versus \textit{Jazz}. Section V presents multinomial classification for the full genre set.
