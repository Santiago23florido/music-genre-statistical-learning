\section{Introduction}
This project studies automatic music genre recognition from acoustic descriptors. The objective is to analyze the structure of the dataset, compare several supervised learning strategies, and select a final model for prediction.

The report is organized as follows. Section II presents the data preparation steps. Section III covers the unsupervised analysis. Section IV focuses on binary classification for \textit{Classical} versus \textit{Jazz}. Section V presents multinomial classification for the full genre set. The report ends with the main conclusions and the final submission choice.
