\section{Part II: Binary Classification}
\subsection{Problem Setup}
This part focuses on the binary discrimination between \textit{Classical} and \textit{Jazz}, the two largest classes in the labeled sample. Restricting the dataset to these classes yields 4,354 labeled observations, with 2,318 Classical tracks and 2,036 Jazz tracks.

\subsection{Logistic Regression Models}
The binary classification stage is naturally formulated with logistic-regression-type models that use the 191 acoustic predictors available in the dataset.

\subsection{Regularized Logistic Regression}
Because the predictor space is large and contains many related descriptor families, regularized logistic regression is a suitable extension for the Classical-versus-Jazz task.
